# Holivaw

Holivaw est un laboratoire personnel de R&D dont l'objectif est de construire des technologies physiques ambitieuses en documentant publiquement l'integralite du processus.

Le premier grand projet du laboratoire est NeuroCore.

## Intention

Holivaw ne cherche pas uniquement a produire un resultat final impressionnant. Le laboratoire cherche a construire une capacite :

- comprendre les briques scientifiques ;
- apprendre les contraintes experimentales ;
- concevoir des prototypes ;
- documenter les echecs ;
- transformer chaque etape en connaissance transmissible.

## Philosophie

- Toujours comprendre avant de construire.
- Toujours construire avant de perfectionner.
- Toujours documenter.
- Ne jamais cacher les echecs.
- Preferer un prototype imparfait a une idee parfaite.

## Role public

La documentation publique et la chaine YouTube doivent montrer la progression reelle du projet.

Chaque video doit laisser une trace tangible :

- prototype ;
- schema ;
- PCB ;
- modele CAD ;
- simulation ;
- publication analysee ;
- experience ;
- achat utile ;
- amelioration de NeuroCore.
