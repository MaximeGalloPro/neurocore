# Feuille de route initiale

Cette feuille de route decrit une trajectoire de depart. Elle doit evoluer avec les apprentissages du projet.

## Phase 0 - Initialiser la memoire du projet

Objectifs :

- creer la structure du depot ;
- documenter la vision ;
- definir les regles de documentation ;
- poser l'architecture generale ;
- preparer le suivi des decisions.

Livrables :

- `README.md` ;
- `docs/` ;
- premier journal de decisions ;
- structure des dossiers techniques.

## Phase 1 - Comprendre le vivant

Objectifs :

- documenter le neurone ;
- comprendre les synapses ;
- expliquer le potentiel d'action ;
- documenter le role des cellules gliales ;
- analyser les organoides.

Livrables possibles :

- notes de lecture ;
- schemas ;
- scripts de videos ;
- bibliographie commentee.

## Phase 2 - Concevoir NeuroCore

Objectifs :

- definir les exigences ;
- comparer plusieurs architectures ;
- identifier les contraintes biologiques, mecaniques, electroniques et logicielles ;
- choisir une premiere architecture prototype.

Livrables possibles :

- document d'architecture ;
- diagrammes ;
- premiers modeles CAD ;
- liste de composants ;
- decisions techniques datees.

## Phase 3 - Interfaces neuronales

Objectifs :

- comprendre les MEA ;
- etudier les microelectrodes ;
- concevoir la chaine d'acquisition ;
- identifier les contraintes de bruit et de filtrage.

Livrables possibles :

- notes techniques ;
- schemas electroniques ;
- prototypes d'amplification ;
- simulations ;
- PCB experimentaux.

## Phase 4 - Construire le laboratoire

Objectifs :

- definir l'environnement minimal ;
- concevoir une premiere chambre de culture ;
- instrumenter temperature, CO2 et humidite ;
- etudier la perfusion.

Livrables possibles :

- architecture de chambre ;
- essais de capteurs ;
- firmware de controle ;
- tableau de risques ;
- protocoles experimentaux.

## Phase 5 - Preparer les cultures cellulaires

Objectifs :

- documenter les pre-requis biologiques ;
- definir les protocoles ;
- identifier les fournisseurs et contraintes legales ;
- reduire les risques avant manipulation.

Livrables possibles :

- protocoles commentes ;
- checklist de securite ;
- bibliographie ;
- plan d'experience.

## Phase 6 - Premiers NeuroCore

Objectifs :

- integrer les premiers sous-systemes ;
- realiser les premieres mesures ;
- documenter les echecs et corrections ;
- stabiliser une premiere version reproductible.

Livrables possibles :

- prototype integre ;
- mesures ;
- journaux d'experience ;
- video de synthese ;
- version taguee du depot.
