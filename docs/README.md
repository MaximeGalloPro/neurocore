# Documentation NeuroCore

Ce dossier contient la memoire technique du projet NeuroCore.

Il doit permettre a une personne exterieure de comprendre :

- pourquoi le projet existe ;
- quelles decisions ont ete prises ;
- quelles alternatives ont ete considerees ;
- quelles experiences ont ete menees ;
- quels resultats ont ete obtenus ;
- quelles erreurs ont ete utiles.

## Organisation

- `vision/` : intention, philosophie et objectifs long terme.
- `architecture/` : architecture generale et sous-systemes.
- `decisions/` : decisions techniques datees et justifiees.
- `documentation-rules.md` : regles de documentation du projet.
- `roadmap.md` : trajectoire de travail initiale.

## Regle de base

Une documentation incomplete mais honnete est preferable a une documentation parfaite jamais ecrite.
