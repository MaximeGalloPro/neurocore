# NeuroCore

NeuroCore est le premier grand projet du laboratoire personnel de R&D Holivaw.

L'objectif long terme est de construire progressivement une plateforme experimentale permettant d'interagir avec un reseau neuronal biologique, en documentant publiquement chaque etape du processus.

Le but n'est pas seulement de construire une machine. Le but est de devenir capable de la construire.

## Principes

- Comprendre avant de construire.
- Construire avant de perfectionner.
- Documenter chaque decision, chaque erreur et chaque progression.
- Preferer un prototype imparfait a une idee parfaite.
- Rendre le projet reproductible, critiquable et ameliorable.

## Sous-systemes principaux

NeuroCore sera construit comme une plateforme modulaire composee de plusieurs sous-systemes :

- reseau neuronal biologique ;
- chambre de culture ;
- perfusion ;
- controle environnemental ;
- interface neuronale ;
- electronique ;
- logiciel ;
- firmware ;
- documentation experimentale.

## Structure du depot

```text
docs/          Documentation generale, architecture, vision, decisions
hardware/      Conception mecanique et systemes physiques
electronics/   Schemas, circuits, PCB, instrumentation
cad/           Modeles CAD et fichiers de conception
software/      Logiciels d'acquisition, visualisation et automatisation
firmware/      Code embarque pour controle et instrumentation
experiments/   Protocoles, resultats, erreurs, observations
videos/        Preparation, scripts et traces des videos YouTube
media/         Images, schemas, rendus et supports visuels
references/    Publications, notes de lecture et bibliographie
```

## Documentation

La documentation est un actif central du projet. Chaque sujet important doit repondre aux questions suivantes :

- Pourquoi ?
- Comment ?
- Quelles alternatives existent ?
- Quels sont les avantages ?
- Quels sont les inconvenients ?
- Comment le reproduire ?
- Quelles publications scientifiques le decrivent ?

## Etat du projet

Version initiale du depot : `0.1`.

Le projet demarre par la constitution de la documentation, l'analyse scientifique des briques necessaires, puis la conception progressive d'une premiere architecture NeuroCore.
