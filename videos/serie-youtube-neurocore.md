# Serie YouTube NeuroCore

## Vision

La chaine YouTube est la vitrine publique du laboratoire Holivaw.

Elle ne sert pas uniquement a divertir ou a vulgariser.

Elle a plusieurs objectifs simultanes :

- documenter publiquement le projet ;
- montrer la progression reelle ;
- demontrer notre rigueur scientifique ;
- attirer une communaute ;
- attirer des collaborateurs ;
- attirer des mecenes et investisseurs ;
- financer progressivement le materiel necessaire au projet.

L'objectif est qu'apres quelques dizaines de videos, quelqu'un puisse avoir confiance dans notre capacite a mener un projet de R&D complexe.

## Principe fondamental

La chaine ne raconte pas un resultat.

Elle raconte une progression.

Nous ne pretendons jamais savoir construire un ordinateur biologique des le depart.

Nous montrons comment nous devenons progressivement capables de le construire.

## Le NeuroCore est compose de plusieurs briques

Chaque brique peut devenir une saison entiere.

Le systeme complet comprend :

- reseau neuronal biologique ;
- chambre de culture sterile ;
- systeme de perfusion ;
- controle environnemental ;
- interface electrophysiologique, MEA ou microelectrodes ;
- electronique d'acquisition ;
- electronique de stimulation ;
- logiciel de controle ;
- analyse des signaux ;
- intelligence artificielle ;
- robotisation du laboratoire ;
- boitier final.

Chaque element peut generer plusieurs videos.

## Proposition actuelle des saisons

Cette proposition est une carte de depart. Elle pourra etre decoupee, fusionnee ou reorganisee selon l'avancement reel du projet.

### Saison 1 - Comprendre le vivant

Objectif :

Comprendre suffisamment les neurosciences pour pouvoir expliquer simplement comment fonctionne un neurone.

Exemples de videos :

- Pourquoi un neurone est-il different d'un transistor ?
- Comment un neurone produit-il un potentiel d'action ?
- Les synapses expliquees simplement.
- Les reseaux neuronaux biologiques.
- Les astrocytes.
- Les cellules souches.
- Les organoides cerebraux.
- Pourquoi les neurones meurent.
- Les limites des cultures cellulaires.

Boss de fin de saison :

Etre capable de simuler un neurone sur ordinateur.

## Format type d'une video

Chaque video suit approximativement la meme structure.

1. Pourquoi cette question est importante.
2. Explication scientifique.
3. Etat de l'art.
4. Comment les laboratoires procedent aujourd'hui.
5. Notre approche.
6. Ce que nous allons construire.
7. Les difficultes attendues.
8. Le resultat obtenu.
9. Ce qu'il reste a faire.

Chaque video doit faire progresser le projet.

## Livrable attendu a chaque episode

Une video ne doit jamais etre uniquement theorique.

Elle doit produire au moins un livrable :

- un schema ;
- une documentation ;
- un prototype ;
- une simulation ;
- une analyse scientifique ;
- un modele CAD ;
- un PCB ;
- un script ;
- une experience ;
- un protocole ;
- une decision d'architecture.

Le projet doit progresser a chaque episode.

## Philosophie editoriale

Nous preferons montrer :

- la reflexion ;
- les erreurs ;
- les compromis ;
- les essais ;
- les decisions techniques ;

plutot que de produire une video spectaculaire sans contenu.

L'objectif est de devenir une reference documentaire sur la construction d'un ordinateur biologique.

## Evolution

Cette feuille de route n'est pas figee.

Les saisons pourront etre decoupees, fusionnees ou reorganisees au fil de l'avancement du projet.

Le depot GitHub constitue la source de verite.

Les videos racontent l'histoire de cette progression.
