# Niveau 2 - Ressources avancees

1. Neuronal Dynamics - Introduction  
   URL : https://neuronaldynamics.epfl.ch/online/index.html  
   Pourquoi : introduit le neurone comme systeme dynamique, pas comme simple interrupteur.

2. MIT OpenCourseWare - Circuits and Electronics  
   URL : https://ocw.mit.edu/courses/6-002-circuits-and-electronics-spring-2007/  
   Pourquoi : ressource solide pour comprendre le vocabulaire electronique de la comparaison.

3. OpenStax Anatomy and Physiology 2e - Nervous Tissue  
   URL : https://openstax.org/books/anatomy-and-physiology-2e/pages/12-2-nervous-tissue  
   Pourquoi : apporte une vision plus anatomique et fonctionnelle du neurone dans le systeme nerveux.
