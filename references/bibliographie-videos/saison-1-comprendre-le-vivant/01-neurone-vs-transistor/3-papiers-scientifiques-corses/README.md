# Niveau 3 - Papiers scientifiques corses

1. Indiveri et Liu - Memory and information processing in neuromorphic systems  
   URL : https://arxiv.org/abs/1506.03264  
   Pourquoi : met en perspective les differences entre calcul electronique, neuromorphique et biologique.

2. A Bibliometric Review of Neuromorphic Computing and Spiking Neural Networks  
   URL : https://arxiv.org/abs/2304.06897  
   Pourquoi : donne une vue d'ensemble des liens entre neurones biologiques, reseaux impulsionnels et materiel neuromorphique.

3. Izhikevich - Which model to use for cortical spiking neurons?  
   URL : https://www.izhikevich.org/publications/whichmod.pdf  
   Pourquoi : compare differents modeles de neurones impulsionnels et montre que le neurone n'est pas un composant logique simple.
