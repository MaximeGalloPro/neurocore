# Niveau 3 - Papiers scientifiques corses

1. Izhikevich - Polychronization: Computation with Spikes  
   URL : https://www.izhikevich.org/publications/spnet.pdf  
   Pourquoi : illustre comment des reseaux de neurones a spikes peuvent produire des structures temporelles complexes.

2. Brette - Philosophy of the spike: rate-based vs spike-based theories of the brain  
   URL : https://arxiv.org/abs/1502.05200  
   Pourquoi : questionne ce que les spikes representent vraiment dans un reseau biologique.

3. Implementing Spiking Neural Networks on Neuromorphic Architectures: A Review  
   URL : https://arxiv.org/abs/2202.08897  
   Pourquoi : revue technique sur les reseaux impulsionnels et leur implementation materielle.
