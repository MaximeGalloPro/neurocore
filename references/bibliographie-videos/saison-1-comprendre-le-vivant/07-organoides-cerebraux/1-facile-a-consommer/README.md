# Niveau 1 - Facile a consommer

1. MedlinePlus - Stem Cells  
   URL : https://medlineplus.gov/stemcells.html  
   Pourquoi : base necessaire avant de comprendre comment on obtient un organoide.

2. Learn.Genetics - Stem Cells  
   URL : https://learn.genetics.utah.edu/content/stemcells/  
   Pourquoi : ressource simple pour comprendre differenciation et origine cellulaire des organoides.

3. OpenStax Biology 2e - Fertilization and Early Embryonic Development  
   URL : https://openstax.org/books/biology-2e/pages/43-6-fertilization-and-early-embryonic-development  
   Pourquoi : aide a comprendre les notions de differenciation et developpement necessaires aux organoides.
