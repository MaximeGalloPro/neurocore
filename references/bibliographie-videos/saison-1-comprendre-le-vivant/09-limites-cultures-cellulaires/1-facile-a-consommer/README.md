# Niveau 1 - Facile a consommer

1. ATCC - Animal Cell Culture Guide  
   URL : https://www.atcc.org/resources/culture-guides/animal-cell-culture-guide  
   Pourquoi : base pratique pour comprendre milieu, contamination, passage et conditions de culture.

2. CDC - Biosafety in Microbiological and Biomedical Laboratories  
   URL : https://www.cdc.gov/labs/BMBL.html  
   Pourquoi : guide gratuit pour comprendre les exigences de securite et les risques de manipulation.

3. WHO - Laboratory Biosafety Manual, 4th edition  
   URL : https://www.who.int/publications/i/item/9789240011311  
   Pourquoi : cadre de base sur la securite biologique et les risques de manipulation.
